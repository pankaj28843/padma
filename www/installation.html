Please file any bugs that you find on our bug reporting tool.

The current stable version of Padma is 0.4.3.

Version 0.4.3 [install]

  • Added: Support for Dainik Bhaskar (Hindi)
  • Added: Ability to configure right click options both for scripts and schemes
  • Added: Ability to set script in which a website should be viewed
  • Fixed: Remove heuristic transform as it is not necessary

Version 0.4.2 [install]

  • Added: Support for Viktan, Kumudam, Shree Tam 0802 and Elango Panchali fonts (Tamil)
  • Added: Support for Subak (Marathi) font (used by Sakaal)
  • Fixed: Dont show manual transform options if no text is selected

Version 0.4.1 [install]

  • Added: Support for TeluguLipi font
  • Added: Support for Kairali, Revathi and Manorama fonts (Malayalam)
  • Added: Support for Transform from ISCII and ITRANS to Malayalam, Tamil and Devanagari
  • Added: Support for Firefox 1.5 beta
  • Added: Support for setting text alignment to left if specified as justify for Unicode transforms
  • Fixed: Broken 'Transform to RTS' option
  • Fixed: In sync with offline transformer at Geocities

Version 0.4.0 [install]

  • Added: Support for Malayalam (ML-TTKarthika)
  • Added: Support for automatically fixing character encoding if enabled by user
  • Added: Preliminary support for Tamil (Vikatan)
