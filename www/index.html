Padma: A Transformer for Telugu Text from RTS To Unicode

Padma is an extension for Mozilla-based applications for transfroming Telugu text written in RTS (Rice Transliteration Scheme) to Unicode. Padma does this transformation by replacing the selected text with its equivalent Unicode representation.

After Padma is installed, two new menu items are displayed whenever the context menu is launched, which is typically when the right mouse button is pressed. Text in the selected region is transfromed from RTS to Unicode. This selection can span arbitrary elements including but not limited to images, tables. The two menu items differ only in specifying the start mode for the transformation - English or Telugu. Following the standard RTS conventions, Padma uses '#' as a delimiter to switch between languages.

Padma is inspired by Suresh Kolichala's implementation of a convertor for the Internet Explorer.
