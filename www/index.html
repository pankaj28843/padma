Padma: A Telugu Text Transformer

Padma is a set of technologies for transfroming Telugu text between various public and proprietary formats. A Mozilla extension (by the name Padma) illustrates the use of the technology for Mozilla based applications. Output formats supported are Unicode and RTS. Currently supported input methods are RTS (Rice Transliteration Scheme), Unicode, the font scheme used by Eenadu, Shree Tel 900 and Hemalatha fonts. Shree Tel 900 is used by many web sites including Andhra Jyothy and Hemalatha - a CDAC developed font is used by Andhra Prabha among others. This extension transforms the selected text to the desired output format.

After Padma is installed, new menu items are displayed whenever the context menu is launched (typically when the right mouse button is pressed). Three menu items transform selected text to Unicode - two of them expect input in RTS and one for all other encoding schemes. Another menu item transforms selected text to RTS. This selection can span arbitrary elements including but not limited to images, tables. For RTS input, the two menu items differ only in specifying the start mode for the transformation - English or Telugu. Following the standard RTS conventions, Padma uses '#' as a delimiter to switch between languages. For RTS output, three different styles are supported - Kanneganti, Pillalamarri and Palana. See here for a definition of the above 3 styles.

The current stable version of Padma is 0.2.6. The current development release is 0.3.3. Support for Eenadu was first added in the 0.3.0 release. Support for Shree Tel 900 (and hence Andhra Jyothy) was first added in the 0.3.1 release. Transformation from Unicode and to RTS was first added in the 0.3.2 release. Support for Hemalatha (and hence Andhra Prabha) was first added in the 0.3.3 release. Click here for Release Notes.

Padma is inspired by Kolichala Suresh's implementation of a convertor from RTS to Unicode for the Internet Explorer. Many thanks to Kolichala Suresh, Indraganti Padma and Ben Bennett for their inputs and advice.
