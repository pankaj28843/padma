Padma: RTS To Unicode Telugu Transformer

Padma is an extension for Mozilla-based applications for transfroming Telugu text written in RTS to Unicode. Padma does this transformation by replacing the selected text with its equivalent Unicode representation.

After Padma is installed, two new menu items are displayed whenever a region is selected and the right mouse button is pressed i.e. whenever the context menu is launched. This selection can span arbitrary elements including but not limited to images, tables etc. Text in the selected region is transfromed to Unicode Telugu. The two menu items differ only in specifying the start mode for the transformation - English or Telugu. Following conventions developed over a period of time, Padma uses '#' as a delimiter to switch between languages.

Padma is inspired by Suresh Kolicala's implementation of a convertor for the Internet Explorer.
