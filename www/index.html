Padma: A Telugu Text Transformer

Padma is an extension for Mozilla-based applications for transfroming Telugu text encoded in various public and proprietary formats to Unicode. Currently supported input methods are RTS (Rice Transliteration Scheme), the font schemes used by Eenadu and Andhra Jyothy news papers. Padma does this transformation by replacing the selected text with its equivalent Unicode representation.

After Padma is installed, new menu items are displayed whenever the context menu is launched, which is typically when the right mouse button is pressed. Two menu items provide support for RTS transformation and one for all other encoding schemes. Text in the selected region is transfromed to Unicode. This selection can span arbitrary elements including but not limited to images, tables. For RTS, the two menu items differ only in specifying the start mode for the transformation - English or Telugu. Following the standard RTS conventions, Padma uses '#' as a delimiter to switch between languages.

The current stable version of Padma is 0.2.6. The current development release is 0.3.1. Support for Eenadu was first added in the 0.3.0 release. Support for Andhra Jyothy was first added in the 0.3.1 release. Click here for Release Notes.

Padma is inspired by Suresh Kolichala's implementation of a convertor from RTS to Unicode for the Internet Explorer. Many thanks to Suresh Kolichala, Padma Indraganti and Ben Bennett for their inputs and advice.
